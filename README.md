# midi-merger


A simple browser-based tool to merge multiple MIDI files into a single type 1 MIDI file with all tracks, tempo changes, and automation
preserved. 


Notes and credits: 
-Safari may have issues with large MIDI files.
- Built using [midi-file](https://www.npmjs.com/package/midi-file) JavaScript library.  
- Inspired by the need to merge Type 0 MIDI files into Type 1 for DAWs like Ableton Live.

Feel free to use, modify, and share!
